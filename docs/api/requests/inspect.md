# inspect

Prints request & response details to the console.

::: tip
- Use it for debugging purpose.
- By default, request and response are printed to the terminal when a test case fails.
:::

## Syntax

```js
inspect()
inspect('path')
```

### âœ…  Correct Usage

await spec()

  .inspect('age');
