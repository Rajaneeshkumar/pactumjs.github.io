# Stash

Stash Options.

## List of Stash Options

| Method                        | Description                           |
| ----------------------------  | ------------------------------------  |
| `addDataTemplate`             | add data template                     |
| `addDataMap`                  | add data map                          |
| `getDataMap`                  | get data map                          |
| `getDataTemplate`             | get data template                     |
| `getDataStore`                | get data store                        |
| `loadData`                    | load data from file system            |
